# nvidia-SDDM-optimus-script
script per installare con SDDM nvidia driver e accellerazione browser base chrome con optimus manuale solo nvidia gpu 

## Installazione 
- chmod +x ./nvidia.sh
- ./nvidia.sh

## Pacchetti installati
- nvidia
- nvidia-utils
- nvidia-settings
- libva-mesa-driver
- libva-vdpau-driver
- libva-utils
